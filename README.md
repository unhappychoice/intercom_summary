# IntercomSummary

## Installation

```ruby
gem 'intercom_summary'
```

or

```sh
$ gem install intercom_summary
```

## Usage

```
INTERCOM_APP_ID=your_app_id INTERCOM_APP_KEY=your_app_key intercom_summary
```
